# JSON Formatter

A utility skill for formatting and validating JSON data.

## Usage

When the user asks to format or validate JSON:

1. Parse the input JSON
2. Validate syntax
3. Pretty-print with proper indentation
4. Report any errors found

## Examples

- "Format this JSON for me"
- "Is this valid JSON?"
- "Pretty print this data"
