# CSV Analyzer

Analyze CSV files to extract insights, compute statistics, and generate visualizations.

## Capabilities

- Parse CSV files with automatic delimiter detection
- Generate summary statistics (mean, median, mode, std dev)
- Identify data quality issues (nulls, duplicates, outliers)
- Create basic visualizations

## Usage

Provide a CSV file path and specify what analysis you want:

- "Analyze this CSV file"
- "Show me statistics for the sales column"
- "Find outliers in the data"

## Examples

### Basic Analysis
```
Analyze /path/to/data.csv
```

### Column Statistics
```
Show statistics for the 'revenue' column in sales.csv
```
